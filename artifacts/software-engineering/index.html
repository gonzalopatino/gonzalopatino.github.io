Artifact Introduction

This artifact focuses on software engineering principles as applied to embedded systems development. It demonstrates modular design, component-based architecture, maintainability, and real-world engineering practices that transform prototype code into production-ready software. The work originates from the thermostat project and showcases the application of professional software engineering methodologies to a FreeRTOS-based embedded system.
